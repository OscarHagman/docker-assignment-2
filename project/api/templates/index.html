Hello and welcome to Oscars tiny-API for Docker demonstration purposes.

To register an account: "/register" - Expected POST request: { "username": , "password": } (You will get 6 tokens when registering)
To store a sentence: "/store" - Expected POST request: { "username": , "password": , "sentence": } (Costs 1 token)
To retrieve stored sentence: "/get" - Expected POST request: { "username": , "password": } (Costs 1 token)
